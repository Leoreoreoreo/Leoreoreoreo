## What I'm building

### AutoEditor — *Audio Edit Automation Pipeline*

A cross-platform .NET 10 pipeline that converts handwritten audio-edit marks into machine-readable cut lists. Integrates Whisper.net speech-to-text with custom alignment logic; iterated to **95.5% token match** on hour-long audio. **400+ xUnit tests**.

> *Public repo coming soon — currently sanitising for release.*

### Heavy Processor — *Internal Audiobook Post-Processing Tool*

Python (Flask) tool used in production for **1+ year**, saving **2–3 hours per title** by replacing a disjointed manual process with file ingest, workflow automation, metadata handling, and delivery prep.

> *Private — internal Heavy Entertainment tool.*

### Heavy Check — *Audiobook QC Workflow Tool*

Python tool orchestrating AWS Transcribe jobs, transcript export, and script PDF preparation for the QC workflow.

> *Private — internal Heavy Entertainment tool.*

### Northcoders Capstone

Full-stack C# / .NET application with ASP.NET Core backend and ASP.NET MVC + Razor frontend.
